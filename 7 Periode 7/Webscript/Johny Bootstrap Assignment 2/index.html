This header indicates that this page contains information about Bootstrap

Bootstrap is a CSS framework, used to increase the responsiveness of websites, especially for mobile devices. It works by putting elements in a grid, consisting of rows and clumns. You can set up a row, and nest columns in it. This nesting can be repeated, by nesting rows within those columns, and then other columns within those rows.
